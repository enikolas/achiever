# Achiever

A tool built to... _Achieve!_

## Dependencies

- node
- npm

## Install

    npm install

## Run

    npm run build && npm run serve

or

    npm start

## Watch

In a terminal window:

    npm run watch

And in another:

    npm run serve

## Open

Go to [localhost:3000](http://localhost:3000).
